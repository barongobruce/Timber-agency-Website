# Riverside Timber Suppliers

A modern, responsive website for **Riverside Timber Suppliers**, a trusted timber and building materials company based in Nairobi, Kenya.

## About

Riverside Timber Suppliers has been providing high-quality timber and wood products since 2010. We specialize in premium timber such as Cypress, Bluegum, Pine, Grevillea, and Treated Timber, along with value-added products including Fascia Boards, TNG Boards, Cornice, Door Frames, Doors, and Fencing Posts (Round Poles).

We also offer reliable on-site delivery services across Nairobi and surrounding areas.

## Features

- Clean and professional design
- Product gallery with multiple images per item
- Google Maps integration
- Customer reviews section
- Contact form connected to WhatsApp
- Fully responsive (mobile-friendly)

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
